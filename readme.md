# 📊 Test_for_effectiveMobile

## 📌 Описание проекта

Этот проект содержит автоматические тесты (UI и API) для демонстрационного стенда [Report Portal](https://demo.reportportal.io).  
Реализованы сценарий добавления виджета через UI, а также добавление дэшборда через API.

---

## ✅ Выполненные тесты

### UI-тест: Добавление нового виджета

**Шаги:**
1. Вход в систему;
2. Переход на страницу с дэшбордами;
3. Открытие первого дэшборда из списка;
4. Проверка количества виджетов до операции по добавленю нового
5. Нажатие кнопки **"Add new widget"**;
6. Выбор типа виджета `"Project activity panel"`  
   ⚠️ Виджет `"Task Progress"` отсутствует в доступных вариантах, поэтому выбран аналогичный доступный тип;
7. Переход к следующему шагу (2 раза);
8. Добавление виджета;
9. Проверка появления окна с сообщением **"Widget has been added successfully"**.
10. Проверка количества виджетов после добавления нового и сравнение с ожидаемым количеством

---

### API-тест: Создание дэшборда

**Позитивный сценарий:**
- Отправка корректного тела запроса с использованием API-ключа
- Проверка, что возвращается код `201 Created`
- Валидация, что возвращаемое тело содержит ID созданного дэшборда
- Удаление созданного дэшборда после выполнения теста

**Негативный сценарий:**
- Отправка запроса без тела
- Проверка кода `400 Bad Request`
- Валидация сообщения об ошибке

## ⚙️ Используемые технологии

- 📌 Язык: **Java 17+**
- 💠 UI: **Selenium WebDriver**
- 🔬 API: **REST Assured**
- ⚖️ Фреймворк: **JUnit 5**
- 💎 Отчётность: **Allure Report**
- 🧪 Сборка: **Maven**
- 📚 Логирование: **SLF4J + Logback**
- 🌐 Браузер: **Chrome, Firefox, Edge**

---

## 🔐 Важно

- **Обновите токен** в файле config.yaml перед проведением тестов API

- **Необходимо установить последние версии браузеров** для корректной работы с драйверами, поставляемыми WebDriverManager

- **Тип виджета `"Task Progress"`** недоступен — заменён на `"Project activity panel"`.

## 🚀 Как запустить

### Тесты

с браузером по умолчанию Chrome

```bash
mvn clean test
```

или с выбором браузера

```bash
mvn clean test -Dbrowser=browsername
```

### Allure-отчёт

```bash
allure serve target/allure-results
```
